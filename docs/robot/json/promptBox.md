# 提示框



## 弹窗提示

**参数说明：**

| 参数                    | 类型 | 说明                            |
|:----------------------|:---|:------------------------------|
|`popup`| `string` |文件树弹窗提醒。|
|`content`| `int` |弹窗提示指令所在文件树的位置。|

**使用示例：**

实现：设置速度比例系数为50%。

```json

{"state":"popup","content":1}

```

**返回示例：**

```json

{"command":"popup","popup_result":true}

```

| 参数                | 类型     | 说明      |
|:------------------|:-------|:----------------------------------------|
|`popup_result`            | `bool` |`ture`：继续执行；flase： |
