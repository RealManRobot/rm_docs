## （2）反馈机械臂关节角度

| 功能描述 | 反馈机械臂关节角度                    |
| :--- | :--------------------------- |
| 参数说明 | `joint_degree`：机械臂关节角度       |
| 命令格式 | 六自由度：{s\:s,s:\[i,i,i,i,i,i]} |

**使用示例：**
说明：反馈机械臂关节角度

六自由度

```json
{"state":"joint_degree","joint":[10,20,30,40,50,60]}  
```

七自由度

```json
{"state":"joint_degree","joint":[10,20,30,40,50,60,70]}
```
