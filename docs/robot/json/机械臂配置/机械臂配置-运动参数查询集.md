# 机械臂配置-运动参数查询集

## 查询机械臂末端最大线速度

查询机械臂末端最大线速度。

**参数说明：**

| 参数                     | 类型 | 说明                       |
| :----------------------- | :--- | :------------------------- |
| `get_arm_max_line_speed` |      | 查询机械臂末端最大线速度。 |

**使用示例：**
实现：查询机械臂末端最大线速度。

```json
{"command":"get_arm_max_line_speed"}
```
