# 机械臂配置-运动参数反馈集

## 反馈机械臂末端最大线速度

反馈机械臂末端最大线速度。

**参数说明：**

| 参数                 | 类型 | 说明                       |
| :------------------- | :--- | :------------------------- |
| `arm_max_line_speed` |      | 反馈机械臂末端最大线速度。 |

**反馈示例：**
实现：反馈机械臂末端最大线速度，0.5m/s，分辨率：0.001m/s。

```json
{"state":"arm_max_line_speed","arm_line_speed":500}
```
