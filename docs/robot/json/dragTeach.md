# 拖动示教

## 获取拖动示教轨迹

**参数说明：**

| 参数                    | 类型 | 说明                            |
|:----------------------|:---|:------------------------------|
|`save_trajectory`| `string` |保存轨迹。|
|`trajectory_name`| `int` |轨迹要保存的名称。|

**使用示例：**

实现：获取刚拖动过的轨迹，在拖动示教后调用。

```json

{"command":"save_trajectory", "trajectory_name":"XXX"}

```

**返回示例：**

```json

{"point":[30971, 56885, -3416, 76201, 47121, -4845]}

```

**参数说明：**
| 参数                | 类型     | 说明      |
|:------------------|:-------|:----------------------------------------|
|`point`            | `list` | 部拖动示教点位。 |
