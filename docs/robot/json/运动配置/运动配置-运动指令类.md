# 运动配置-运动指令类

## 轨迹急停

轨迹急停。

**参数说明：**

| 参数           | 类型 | 说明       |
| :------------- | :--- | :--------- |
| `set_arm_stop` |      | 姿态步进。 |

**使用示例：**
实现：轨迹急停。

```json
{"command":"set_arm_stop"}
```

**返回示例：**

```json
{"command":"set_arm_stop","arm_stop":true}
```

| 参数       | 类型   | 说明                                  |
| :--------- | :----- | :------------------------------------ |
| `arm_stop` | `bool` | `ture`：设置成功；`false`：设置失败。 |
